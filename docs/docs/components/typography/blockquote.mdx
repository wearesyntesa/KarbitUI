# Blockquote

A styled quotation block for highlighting cited content.

<TabItem value="html" label="HTML" default>

## Usage

<Demo codes={{ html: usageCode, react: reactCode, vue: vueCode, svelte: svelteCode }}>
  <BrowserOnly>
    {() => {
      require('@wearesyntesa/karbit-ui/define');
      return <div dangerouslySetInnerHTML={{ __html: usageCode }} />;
    }}
  </BrowserOnly>
</Demo>

## Props

| Property | Attribute | Type | Default | Description |
|---|---|---|---|---|
| `variant` | `variant` | `BlockquoteVariant` | `'default'` | Visual variant. |
| `colorScheme` | `color-scheme` | `ColorScheme` | — | Accent color for the left border. |

## Slots

| Name | Description |
|---|---|
| (default) | Default content |
| `cite` | Citation text displayed below the body. |
