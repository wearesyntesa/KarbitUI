## Props

| Property | Attribute | Type | Default | Description |
|---|---|---|---|---|
| `level` | `level` | `HeadingLevel` | `'2'` | Semantic heading level (`'1'`–`'6'`). Determines the rendered `<h1>`–`<h6>` element and default size. |
| `size` | `size` | `HeadingSize` | — | Explicit size override. When unset, derived from `level` (e.g. level `'1'` → `'4xl'`). |
| `weight` | `weight` | `HeadingWeight` | `'bold'` | Font weight. |
| `tone` | `tone` | `HeadingTone` | — | Semantic color tone. When unset, uses the recipe's default primary tone. |
| `truncate` | `truncate` | `boolean` | `false` | Truncate overflowing text with an ellipsis on a single line. |

## Slots

| Name | Description |
|---|---|
| (default) | Default content |
