## Usage

    {() => {

## Props

| Property | Attribute | Type | Default | Description |
|---|---|---|---|---|
| `block` | `block` | `boolean` | `false` | Display as a full-width block with copy button. When `false`, renders as inline `<code>`. |
| `language` | `language` | `string` | `""` | Language identifier for syntax highlighting (e.g. `'typescript'`, `'html'`). Requires `KbCode.highlighter` to be set. |
| `filename` | `filename` | `string` | `""` | File name displayed in a header bar above block code. Only visible when `block` is true. |

## Slots

| Name | Description |
|---|---|
| (default) | Default content |
