## Props

| Property | Attribute | Type | Default | Description |
|---|---|---|---|---|
| `variant` | `variant` | `TextVariant` | `'body'` | Typography preset controlling font family, case, and tracking. |
| `size` | `size` | `FontSizeValue` | — | Font size override. When variant is `'body'`, defaults to `'base'`; other variants use their recipe size. |
| `tone` | `tone` | `TextTone` | — | Semantic color tone. Falls back to the variant's default tone when unset. |
| `clamp` | `clamp` | `number` | — | Maximum visible lines before truncation with an ellipsis (1–6). |
| `noWrap` | `no-wrap` | `boolean` | `false` | Prevent text wrapping with `whitespace-nowrap`. |
| `truncate` | `truncate` | `boolean` | `false` | Truncate overflowing text with an ellipsis on a single line. |
| `as` | `as` | `TextAs` | `'p'` | HTML element to render. |
