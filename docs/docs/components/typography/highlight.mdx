## Props

| Property | Attribute | Type | Default | Description |
|---|---|---|---|---|
| `text` | `text` | `string` | `""` | Full text to render when using `query` mode. |
| `query` | `query` | `string` | `""` | Substring to highlight within `text`. Case-insensitive. |
| `colorScheme` | `color-scheme` | `ColorScheme` | — | Background color scheme. |

## Slots

| Name | Description |
|---|---|
| (default) | Default content |
