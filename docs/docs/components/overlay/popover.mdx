## Props

| Property | Attribute | Type | Default | Description |
|---|---|---|---|---|
| `placement` | `placement` | `PopoverPlacement` | `'bottom'` | Position of the popover relative to the trigger element. |
| `trigger` | `trigger` | `PopoverTrigger` | `'click'` | How the popover is activated - `'click'` toggles on click, `'hover'` on mouse enter/leave. |
| `size` | `size` | `PopoverSize` | `'sm'` | Popover panel width and padding size. |
| `open` | `open` | `boolean` | `false` | Programmatically control visibility. Reflected to attribute. |
| `closeOnEscape` | `close-on-escape` | `boolean` | `true` | Close the popover when the Escape key is pressed. |
| `closeOnOutside` | `close-on-outside` | `boolean` | `true` | Close the popover when clicking outside of it. |
| `closable` | `closable` | `boolean` | `false` | Show a close button in the header area. |
| `showArrow` | `show-arrow` | `boolean` | `false` | Render a small arrow pointing toward the trigger element. |
| `offset` | `offset` | `number` | `2` | Distance between the popover and the trigger (0–4 Tailwind spacing units). |
| `openDelay` | `open-delay` | `number` | `0` | Delay in ms before showing the popover on hover trigger. |
| `closeDelay` | `close-delay` | `number` | `150` | Delay in ms before hiding the popover after mouse leaves (hover trigger). |
| `autoFocus` | `auto-focus` | `boolean` | `false` | Automatically focus the first focusable element inside the popover when opened. |
| `trapFocus` | `trap-focus` | `boolean` | `false` | Trap keyboard focus within the popover while open. |
| `ariaLabel` | `aria-label` | `string \| null` | `null` | Accessible label for the dialog when no visible header is provided. |

## Slots

| Name | Description |
|---|---|
| (default) | Default content |
| `trigger` | Element that opens the popover. |
| `header` | Optional popover header. |
| `footer` | Optional popover footer (only renders when provided). |

## Events

| Event | Detail | Description |
|---|---|---|
| `kb-open` | `{ source?: string }` or none | Dispatched when popover opens. |
| `kb-close` | `{ source?: string }` or none | Dispatched when popover closes (after exit animation). |

## Accessibility

### Keyboard Navigation

| Key | Action |
|---|---|
| `Enter` | Activate or select the focused item |
| `Escape` | Close or cancel the current action |
| `Space` | Activate or select the focused item |
| `Tab` | Move focus to the next focusable element |

### ARIA

**Roles:** `dialog`

**Attributes:** `aria-expanded`, `aria-haspopup`, `aria-label`, `aria-labelledby`, `aria-modal`
