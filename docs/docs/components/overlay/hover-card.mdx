## Props

| Property | Attribute | Type | Default | Description |
|---|---|---|---|---|
| `placement` | `placement` | `HoverCardPlacement` | `'bottom'` | Position of the card relative to the trigger. |
| `size` | `size` | `HoverCardSize` | `'md'` | Card width preset. |
| `openDelay` | `open-delay` | `number` | `300` | Delay in ms before showing the card. |
| `closeDelay` | `close-delay` | `number` | `200` | Delay in ms before hiding the card. |
| `open` | `open` | `boolean` | `false` | Programmatically control visibility. |

## Slots

| Name | Description |
|---|---|
| (default) | Default content |
| `content` | Card body content. |

## Events

| Event | Detail | Description |
|---|---|---|
| `kb-open` | `{ source?: string }` or none | Dispatched when card becomes visible. |
| `kb-close` | `{ source?: string }` or none | Dispatched when card is hidden. |
