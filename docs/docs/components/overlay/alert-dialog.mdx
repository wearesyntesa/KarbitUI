## Props

| Property | Attribute | Type | Default | Description |
|---|---|---|---|---|
| `closeOnOverlay` | `close-on-overlay` | `boolean` | `false` | Disable overlay click dismiss (default for alert dialogs). |
| `closeOnEscape` | `close-on-escape` | `boolean` | `false` | Disable Escape key dismiss (default for alert dialogs). |
| `closable` | `closable` | `boolean` | `false` | Hide the close button by default for alert dialogs. |

## Slots

| Name | Description |
|---|---|
| (default) | Default content |
| `header` | Dialog title. |
| `footer` | Confirm/cancel buttons. |

## Events

| Event | Detail | Description |
|---|---|---|
| `kb-confirm` | `{ source: "alert-dialog" }` | Dispatched when the confirm action is triggered. |
| `kb-cancel` | `{ source: "alert-dialog" }` | Dispatched when the cancel action is triggered. |
| `kb-open` | `{ source?: string }` or none | Dispatched when dialog opens. |
| `kb-close` | `{ source?: string }` or none | Dispatched when dialog closes (after exit animation). |

## Accessibility

### Keyboard Navigation

| Key | Action |
|---|---|
| `Escape` | Close the overlay and restore focus |
| `Tab` | Cycle focus within the overlay (focus trap) |

### ARIA

**Roles:** `alertdialog`

**Attributes:** `aria-label`, `aria-labelledby`, `aria-modal`

### Focus Management

This component extends `KbOverlayBase` and provides:
- Focus trapping within the overlay when open
- Focus restoration to the trigger element on close
- `Escape` key dismissal
- Sibling elements marked as `inert` while open
- Body scroll locking
