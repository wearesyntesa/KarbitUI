# Drawer

A sliding panel that overlays from the edge of the viewport.

## Props

| Property | Attribute | Type | Default | Description |
|---|---|---|---|---|
| `placement` | `placement` | `DrawerPlacement` | `'right'` | Edge of the viewport from which the drawer slides in. |

## Slots

| Name | Description |
|---|---|
| (default) | Default content |
| `header` | Drawer header content. |
| `footer` | Drawer footer content (only renders when provided). |

## Events

| Event | Detail | Description |
|---|---|---|
| `kb-open` | `{ source?: string }` or none | Dispatched when drawer finishes opening. |
| `kb-close` | `{ source?: string }` or none | Dispatched when drawer is dismissed (after exit animation). |

## Accessibility

### Keyboard Navigation

| Key | Action |
|---|---|
| `Escape` | Close the overlay and restore focus |
| `Tab` | Cycle focus within the overlay (focus trap) |

### ARIA

**Roles:** `dialog`

**Attributes:** `aria-label`, `aria-labelledby`, `aria-modal`

### Focus Management

This component extends `KbOverlayBase` and provides:
- Focus trapping within the overlay when open
- Focus restoration to the trigger element on close
- `Escape` key dismissal
- Sibling elements marked as `inert` while open
- Body scroll locking
