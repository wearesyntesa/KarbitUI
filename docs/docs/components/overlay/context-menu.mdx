## Props

| Property | Attribute | Type | Default | Description |
|---|---|---|---|---|
| `size` | `size` | `ComponentSize` | `'md'` | Menu sizing preset. |

## Slots

| Name | Description |
|---|---|
| (default) | Default content |
| `trigger` | Area that listens for `contextmenu` events. |

## Events

| Event | Detail | Description |
|---|---|---|
| `kb-select` | `{ value: string }` | Dispatched when a menu item is selected. |
| `kb-open` | `{ source?: string }` or none | Dispatched when the menu opens. |
| `kb-close` | `{ source?: string }` or none | Dispatched when the menu closes (after exit animation). |

## Accessibility

### Keyboard Navigation

| Key | Action |
|---|---|
| `ArrowDown` | Move focus to the next item |
| `ArrowUp` | Move focus to the previous item |
| `End` | Move focus to the last item |
| `Enter` | Activate or select the focused item |
| `Escape` | Close or cancel the current action |
| `Home` | Move focus to the first item |
| `Space` | Activate or select the focused item |

### ARIA

**Roles:** `menu`
