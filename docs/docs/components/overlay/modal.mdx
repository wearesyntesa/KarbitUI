## Props

| Property | Attribute | Type | Default | Description |
|---|---|---|---|---|
| `placement` | `placement` | `ModalPlacement` | `'center'` | Vertical positioning of the dialog within the viewport. |

## Slots

| Name | Description |
|---|---|
| (default) | Default content |
| `header` | Modal header with title. |
| `footer` | Modal footer with actions (only renders when provided). |

## Events

| Event | Detail | Description |
|---|---|---|
| `kb-open` | `{ source?: string }` or none | Dispatched when modal finishes opening. |
| `kb-close` | `{ source?: string }` or none | Dispatched when modal is dismissed (after exit animation). |

## Accessibility

### Keyboard Navigation

| Key | Action |
|---|---|
| `Escape` | Close the overlay and restore focus |
| `Tab` | Cycle focus within the overlay (focus trap) |

### ARIA

**Roles:** `dialog`

**Attributes:** `aria-label`, `aria-labelledby`, `aria-modal`

### Focus Management

This component extends `KbOverlayBase` and provides:
- Focus trapping within the overlay when open
- Focus restoration to the trigger element on close
- `Escape` key dismissal
- Sibling elements marked as `inert` while open
- Body scroll locking
