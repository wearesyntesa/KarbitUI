## Props

| Property | Attribute | Type | Default | Description |
|---|---|---|---|---|
| `label` | `label` | `string` | `""` | Text content shown inside the tooltip. No tooltip renders when empty. |
| `placement` | `placement` | `TooltipPlacement` | `'top'` | Position of the tooltip relative to the trigger element. |
| `size` | `size` | `TooltipSize` | `'sm'` | Tooltip text and padding size. |
| `variant` | `variant` | `TooltipVariant` | `'dark'` | Color variant - `'dark'` (inverted) or `'light'` (surface-colored). |
| `disabled` | `disabled` | `boolean` | `false` | Prevents the tooltip from showing. |
| `open` | `open` | `boolean` | `false` | Programmatically control visibility. Reflected to attribute. |
| `showArrow` | `show-arrow` | `boolean` | `true` | Render a small arrow pointing toward the trigger element. |
| `offset` | `offset` | `number` | `2` | Distance between the tooltip and the trigger (0–4 Tailwind spacing units). |
| `openDelay` | `open-delay` | `number` | `200` | Delay in ms before showing the tooltip on hover. |
| `closeDelay` | `close-delay` | `number` | `0` | Delay in ms before hiding the tooltip after mouse leaves. |

## Slots

| Name | Description |
|---|---|
| (default) | Default content |

## Events

| Event | Detail | Description |
|---|---|---|
| `kb-open` | `{ source?: string }` or none | Dispatched when tooltip becomes visible. |
| `kb-close` | `{ source?: string }` or none | Dispatched when tooltip is hidden (after exit animation). |

## Accessibility

### Keyboard Navigation

| Key | Action |
|---|---|
| `Escape` | Close or cancel the current action |

### ARIA

**Roles:** `tooltip`

**Attributes:** `aria-describedby`, `aria-label`
