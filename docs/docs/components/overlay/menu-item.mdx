## Props

| Property | Attribute | Type | Default | Description |
|---|---|---|---|---|
| `value` | `value` | `string` | `""` | Value emitted in the `kb-select` event detail. |
| `disabled` | `disabled` | `boolean` | `false` | Disable interaction on this item. |
| `destructive` | `destructive` | `boolean` | `false` | Apply destructive/danger styling (red text). |

## Slots

| Name | Description |
|---|---|
| (default) | Default content |

## Events

| Event | Detail | Description |
|---|---|---|
| `kb-select` | `{ value: string }` | Dispatched when the item is activated (click, Enter, or Space). |

## Accessibility

### Keyboard Navigation

| Key | Action |
|---|---|
| `Enter` | Activate or select the focused item |
| `Space` | Activate or select the focused item |

### ARIA

**Roles:** `menuitem`

**Attributes:** `aria-disabled`
