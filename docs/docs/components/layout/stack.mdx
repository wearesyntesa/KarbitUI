# Stack

## Props

| Property | Attribute | Type | Default | Description |
|---|---|---|---|---|
| `direction` | `direction` | `Orientation` | `'vertical'` | Stack direction. `'vertical'` stacks top-to-bottom, `'horizontal'` left-to-right. |
| `spacing` | `spacing` | `SpacingValue` | — | Gap between children as a Tailwind spacing value (e.g. `'4'`, `'8'`). |
| `divider` | `divider` | `boolean` | `false` | Insert `<kb-divider>` elements between children automatically. |

## Slots

| Name | Description |
|---|---|
| (default) | Default content |
