## Props

| Property | Attribute | Type | Default | Description |
|---|---|---|---|---|
| `columns` | `columns` | `string` | — | Number of grid columns (e.g. `'3'`, `'none'`). Maps to the `gridCols` style prop. |
| `rows` | `rows` | `string` | — | Number of grid rows. Maps to the `gridRows` style prop. |

## Slots

| Name | Description |
|---|---|
| (default) | Default content |
