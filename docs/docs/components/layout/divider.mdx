## Props

| Property | Attribute | Type | Default | Description |
|---|---|---|---|---|
| `orientation` | `orientation` | `Orientation` | `'horizontal'` | Divider axis. `'horizontal'` draws a top border, `'vertical'` a left border. |
| `dividerColor` | `divider-color` | `ColorValue` | — | Custom border color as a Tailwind color value (e.g. `'blue-500'`). Falls back to theme border. |
| `variant` | `variant` | `DividerVariant` | `'solid'` | Border style variant. |
| `thickness` | `thickness` | `DividerThickness` | `'thin'` | Border thickness level. |
| `label` | `label` | `string` | `""` | Optional inline label text (e.g. `'OR'`). When set, the divider renders as two lines flanking the label. |
| `animated` | `animated` | `boolean` | `false` | Animate in with a scale/opacity transition on mount. |

## Accessibility

### ARIA

**Roles:** `separator`

**Attributes:** `aria-label`, `aria-orientation`
