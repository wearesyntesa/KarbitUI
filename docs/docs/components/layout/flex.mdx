## Props

| Property | Attribute | Type | Default | Description |
|---|---|---|---|---|
| `direction` | `direction` | `FlexDirectionValue` | — | Flex direction. Maps to the `flexDir` style prop. |
| `align` | `align` | `FlexAlignValue` | — | Cross-axis alignment. Maps to the `alignItems` style prop. |
| `justify` | `justify` | `FlexJustifyValue` | — | Main-axis alignment. Maps to the `justifyContent` style prop. |
| `wrap` | `wrap` | `FlexWrapValue` | — | Flex wrapping behavior. Maps to the `flexWrap` style prop. |

## Slots

| Name | Description |
|---|---|
| (default) | Default content |
