## Usage

<Demo codes={{ html: usageCode, react: reactCode, vue: vueCode, svelte: svelteCode }}>
  <BrowserOnly>
    {() => {
      require('@wearesyntesa/karbit-ui/define');

    }}
  </BrowserOnly>
</Demo>

## Props

| Property | Attribute | Type | Default | Description |
|---|---|---|---|---|
| `size` | `size` | `ContainerSize` | — | Named max-width preset. Takes precedence over `max` when set. |
| `max` | `max` | `DimensionValue` | — | Custom Tailwind max-width value (e.g. `'4xl'`, `'prose'`). Ignored if `size` is set. |
| `center` | `center` | `boolean` | `true` | Horizontally center the container with `mx-auto`. |

## Slots

| Name | Description |
|---|---|
| (default) | Default content |
