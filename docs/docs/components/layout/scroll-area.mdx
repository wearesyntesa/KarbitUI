## Props

| Property | Attribute | Type | Default | Description |
|---|---|---|---|---|
| `direction` | `direction` | `ScrollDirection` | `'vertical'` | Scroll direction. |
| `scrollbar` | `scrollbar` | `ScrollbarVisibility` | `'auto'` | Scrollbar visibility mode. |
| `bordered` | `bordered` | `boolean` | `false` | Apply a thin border around the scroll container. |

## Slots

| Name | Description |
|---|---|
| (default) | Default content |

## Events

| Event | Detail | Description |
|---|---|---|
| `kb-scroll` | `{ scrollTop: number, scrollLeft: number }` | Dispatched on scroll with current scroll position. |
