# Components

KarbitUI provides **78 web components** organized into 7 categories. All components are built on [Lit v3](https://lit.dev/) with Light DOM rendering and [Tailwind CSS](https://tailwindcss.com/) styling.

## Architecture

### Base element

All components extend `KbBaseElement`, which provides:

- **Light DOM rendering** — `createRenderRoot()` returns `this` (no Shadow DOM)
- **Style props** — `p`, `m`, `bg`, `color`, `w`, `h`, etc. map to Tailwind classes
- **Class passthrough** — `class` attribute for direct Tailwind class application
- **Event emission** — `emit()` helper for typed `CustomEvent` dispatch
- **Class composition** — `buildClasses()` and `cx()` utilities

### Sizing

Most components support `size` prop with values: `xs`, `sm`, `md` (default), `lg`, `xl`.

### Color schemes

Interactive components support `color-scheme` prop: `blue` (default), `red`, `green`, `yellow`, `gray`, and more.

### Variants

Many components support style variants: `solid`, `outline`, `ghost`, `subtle`, `plain`.

## Import patterns

### Web components (any framework)

```ts
// Register all components
import "@wearesyntesa/karbit-ui/define";

// Or register individually
import "@wearesyntesa/karbit-ui/button/define";
```

### React wrappers

```tsx
// Barrel import
import { Button, Input, Card } from "@wearesyntesa/karbit-ui/react";

// Individual imports (better tree-shaking)
import { Button } from "@wearesyntesa/karbit-ui/react/button";
```

### Types

```ts
import type { ComponentSize, ColorScheme } from "@wearesyntesa/karbit-ui/types";
import type { KbChangeDetail } from "@wearesyntesa/karbit-ui/events";
```

## Event system

Components emit typed `CustomEvent`s with `kb-` prefix:

| Event | Components | Detail |
|---|---|---|
| `kb-change` | Input, Select, Checkbox, etc. | `&#123; value, previousValue &#125;` |
| `kb-click` | Button, Card, Tag | `&#123; originalEvent &#125;` |
| `kb-toggle` | Accordion, Collapsible, Switch | `&#123; open &#125;` |
| `kb-close` | Alert, Modal, Drawer, Toast | `&#123; reason &#125;` |
| `kb-select` | Tabs, Select, Combobox | `&#123; value, label &#125;` |
