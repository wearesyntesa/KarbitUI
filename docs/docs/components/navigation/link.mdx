## Props

| Property | Attribute | Type | Default | Description |
|---|---|---|---|---|
| `href` | `href` | `string` | `'#'` | URL the link navigates to. |
| `variant` | `variant` | `LinkVariant` | `'underline'` | Visual variant controlling underline and hover behavior. |
| `size` | `size` | `ComponentSize` | `'md'` | Text and icon size. |
| `external` | `external` | `boolean` | `false` | Open link in a new tab with `rel="noopener noreferrer"` and show an external icon. |
| `disabled` | `disabled` | `boolean` | `false` | Disables the link, preventing navigation and applying reduced opacity. |
| `truncate` | `truncate` | `boolean` | `false` | Truncate link text with ellipsis when it overflows. |
| `showVisited` | `show-visited` | `boolean` | `false` | Apply visited-link color styling. |
| `colorScheme` | `color-scheme` | `ColorScheme` | — | Color scheme for text, decoration, and hover effects. Overrides the default blue. |

| Name | Description |
|---|---|
| (default) | Default content |
| `icon-left` | Leading icon (SVG auto-sized per `size`). |
| `icon-right` | Trailing icon (SVG auto-sized per `size`). |
