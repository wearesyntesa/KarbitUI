# Breadcrumb

A navigation trail showing the current page location within a hierarchy.

## Props

| Property | Attribute | Type | Default | Description |
|---|---|---|---|---|
| `items` | `items` | `BreadcrumbItem[]` | `[]` | Array of breadcrumb items. The last item is treated as the current page. |
| `separator` | `separator` | `string` | `'chevron'` | Separator between items - a preset name (`'chevron'`, `'slash'`, `'arrow'`, `'dot'`, `'pipe'`) or any custom string. |
| `size` | `size` | `ComponentSize` | `'md'` | Text and gap size. |
| `maxItems` | `max-items` | `number` | `0` | Maximum visible items before collapsing middle items behind an ellipsis button. `0` disables collapsing. |

## Events

| Event | Detail | Description |
|---|---|---|
| `kb-navigate` | `{ index: number, item: BreadcrumbItem }` | Breadcrumb item clicked. Detail: `{ index: number; item: BreadcrumbItem }`. |

## Accessibility

### ARIA

**Attributes:** `aria-current`, `aria-label`
