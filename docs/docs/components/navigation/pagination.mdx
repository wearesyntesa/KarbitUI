## Props

| Property | Attribute | Type | Default | Description |
|---|---|---|---|---|
| `total` | `total` | `number` | `1` | Total number of pages. |
| `page` | `page` | `number` | `1` | Current active page (1-based). |
| `siblings` | `siblings` | `number` | `1` | Number of sibling pages to show around the current page. |
| `size` | `size` | `PaginationSize` | `'md'` | Component size. |

## Events

| Event | Detail | Description |
|---|---|---|
| `kb-page-change` | `{ page: number }` | Fired when the active page changes. `detail.page` is the 1-based page number. |
