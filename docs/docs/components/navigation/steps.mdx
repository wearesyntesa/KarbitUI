## Props

| Property | Attribute | Type | Default | Description |
|---|---|---|---|---|
| `current` | `current` | `number` | `0` | Zero-based index of the current step. |
| `orientation` | `orientation` | `Orientation` | `'horizontal'` | Layout direction. |
| `size` | `size` | `ComponentSize` | `'md'` | Step indicator size. |

## Slots

| Name | Description |
|---|---|
| (default) | Default content |

## Events

| Event | Detail | Description |
|---|---|---|
| `kb-step-click` | `{ index: number }` | Dispatched when a completed step is clicked, with `detail.index`. |
