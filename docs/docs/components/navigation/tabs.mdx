## Props

| Property | Attribute | Type | Default | Description |
|---|---|---|---|---|
| `active` | `active` | `number` | `0` | Zero-based index of the currently active tab. |
| `variant` | `variant` | `TabsVariant` | `'line'` | Visual variant controlling tab styling - `'line'` (indicator bar), `'enclosed'` (bordered tabs), `'solid'` (filled background), or `'unstyled'`. |
| `size` | `size` | `TabsSize` | `'md'` | Tab text and padding size. |
| `colorScheme` | `color-scheme` | `KnownColorScheme` | `'blue'` | Accent color scheme for active tab indicator and text. |
| `fitted` | `fitted` | `boolean` | `false` | Stretch tabs to fill the available width equally. |
| `orientation` | `orientation` | `TabsOrientation` | `'horizontal'` | Tab list layout direction. |
| `align` | `align` | `TabsAlign` | `'start'` | Horizontal/vertical alignment of tabs within the tab list. |
| `disabledTabs` | `disabled-tabs` | `number[]` | `[]` | Array of zero-based tab indices that should be disabled. Attribute accepts comma-separated numbers. |
| `label` | `label` | `string` | `""` | Accessible label for the tab list (required when no visible heading labels the group). |

## Events

| Event | Detail | Description |
|---|---|---|
| `kb-tab-change` | `{ index: number, previousIndex: number }` | Dispatched when active tab changes, with `detail.index` and `detail.previousIndex`. |

## Accessibility

### Keyboard Navigation

| Key | Action |
|---|---|
| `ArrowDown` | Move focus to the next item |
| `ArrowLeft` | Move focus backward or decrease value |
| `ArrowRight` | Move focus forward or increase value |
| `ArrowUp` | Move focus to the previous item |
| `End` | Move focus to the last item |
| `Home` | Move focus to the first item |

### ARIA

**Roles:** `tab`, `tablist`, `tabpanel`

**Attributes:** `aria-controls`, `aria-disabled`, `aria-label`, `aria-labelledby`, `aria-orientation`, `aria-selected`
