## Props

| Property | Attribute | Type | Default | Description |
|---|---|---|---|---|
| `active` | `active` | `number` | `0` | Zero-based index of the currently visible slide. |
| `autoplay` | `autoplay` | `boolean` | `false` | Enable automatic slide advancement. |
| `interval` | `interval` | `number` | `5000` | Autoplay interval in milliseconds. |
| `loop` | `loop` | `boolean` | `true` | Wrap from last slide to first (and vice versa). |
| `showArrows` | `show-arrows` | `boolean` | `true` | Show previous/next arrow buttons in the control bar. |
| `showDots` | `show-dots` | `boolean` | `true` | Show dot indicators in the control bar. |
| `showCounter` | `show-counter` | `boolean` | `true` | Show "1 / N" slide counter in the control bar. |
| `pauseOnHover` | `pause-on-hover` | `boolean` | `true` | Pause autoplay when hovering over the carousel. |
| `swipeable` | `swipeable` | `boolean` | `true` | Enable touch/pointer drag to swipe between slides. |

## Events

| Event | Detail | Description |
|---|---|---|
| `kb-slide-change` | `{ index: number, previousIndex: number }` | Dispatched when the active slide changes, with `detail.index` and `detail.previousIndex`. |
