## Props

| Property | Attribute | Type | Default | Description |
|---|---|---|---|---|
| `items` | `items` | `readonly DataListItem[]` | `[]` | Array of key-value pairs to render. |
| `orientation` | `orientation` | `DataListOrientation` | `'horizontal'` | Layout direction for each item pair. |
| `size` | `size` | `ComponentSize` | `'md'` | Sizing preset. |
| `divider` | `divider` | `boolean` | `false` | Show a divider between items. |

## Slots

| Name | Description |
|---|---|
| (default) | Default content |
