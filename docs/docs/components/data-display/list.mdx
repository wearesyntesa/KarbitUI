## Props

| Property | Attribute | Type | Default | Description |
|---|---|---|---|---|
| `variant` | `variant` | `ListVariant` | `'plain'` | List variant - `'plain'` (unordered) or `'ordered'` (monospace-numbered items). |
| `spacing` | `spacing` | `ListSpacing` | `'normal'` | Vertical spacing between items. |
| `dividers` | `dividers` | `boolean` | `false` | Show border separators between items. |
| `interactive` | `interactive` | `boolean` | `false` | Make all child items hoverable, focusable, and clickable. |
| `emptyText` | `empty-text` | `string` | `'No results'` | Text shown when the list has no items. |
