## Props

| Property | Attribute | Type | Default | Description |
|---|---|---|---|---|
| `items` | `items` | `TimelineItem[]` | `[]` | Timeline items to display. |
| `variant` | `variant` | `TimelineVariant` | `'left'` | Layout variant. |
| `size` | `size` | `ComponentSize` | `'md'` | Size of the timeline. |
| `lineVariant` | `line-variant` | `TimelineLineVariant` | `'solid'` | Line style variant. |
| `showIcons` | `show-icons` | `boolean` | `false` | Show status icons inside indicator squares. |

## Events

| Event | Detail | Description |
|---|---|---|
| `kb-timeline-item-click` | `{ index: number, id: string }` | Dispatched when a timeline item is clicked, with `detail.index` and `detail.id`. |
