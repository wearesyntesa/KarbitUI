## Props

| Property | Attribute | Type | Default | Description |
|---|---|---|---|---|
| `src` | `src` | `string` | — | Image URL. Falls back to initials or icon when unset or broken. |
| `name` | `name` | `string` | — | Full name used to derive initials when no image is available. |
| `size` | `size` | `AvatarSize` | `'md'` | Avatar dimensions. |
| `colorScheme` | `color-scheme` | `ColorScheme` | — | Background color scheme. When unset, a neutral gray is used. |
| `badge` | `badge` | `ColorScheme` | — | Show a small status badge in the top-right corner with the given color scheme. |

## Slots

| Name | Description |
|---|---|
| `icon` | Custom fallback icon rendered when no image or name is available. |
