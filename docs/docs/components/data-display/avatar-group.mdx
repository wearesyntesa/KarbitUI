## Props

| Property | Attribute | Type | Default | Description |
|---|---|---|---|---|
| `max` | `max` | `number` | `0` | Maximum number of visible avatars before showing "+N". 0 = no limit. |
| `size` | `size` | `ComponentSize` | `'md'` | Size propagated to child avatars. |
| `spacing` | `spacing` | `AvatarGroupSpacing` | `'md'` | Overlap spacing between avatars. |
