## Props

| Property | Attribute | Type | Default | Description |
|---|---|---|---|---|
| `variant` | `variant` | `CardVariant` | `'outline'` | Visual variant controlling border, background, and elevation. |
| `size` | `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | Inner padding size. |
| `interactive` | `interactive` | `boolean` | `false` | When true, the card is focusable and emits `kb-click` on activation. |
| `label` | `label` | `string` | `""` | Accessible label for an interactive card (used as `aria-label` on the `role="button"` element). |
| `href` | `href` | `string` | `""` | URL to navigate to - renders the card as an anchor element. |
| `target` | `target` | `string` | `""` | Link target attribute, e.g. `'_blank'`. Only used when `href` is set. |

## Slots

| Name | Description |
|---|---|
| (default) | Default content |
| `header` | Card header content. |
| `footer` | Card footer content. |

## Events

| Event | Detail | Description |
|---|---|---|
| `kb-click` | none | Dispatched when an interactive card is clicked or activated via keyboard. |

## Accessibility

### Keyboard Navigation

| Key | Action |
|---|---|
| `Enter` | Activate or select the focused item |
| `Space` | Activate or select the focused item |

### ARIA

**Roles:** `button`

**Attributes:** `aria-label`
