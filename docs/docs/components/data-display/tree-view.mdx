## Props

| Property | Attribute | Type | Default | Description |
|---|---|---|---|---|
| `nodes` | `nodes` | `readonly TreeNode[]` | `[]` | Array of tree node data to render. |
| `size` | `size` | `ComponentSize` | `'md'` | Controls text and padding sizing. |
| `selectable` | `selectable` | `boolean` | `true` | Allow node selection. |

## Events

| Event | Detail | Description |
|---|---|---|
| `kb-node-toggle` | `{ id: string, expanded: boolean }` | When a node is expanded or collapsed. |
| `kb-node-select` | `{ id: string, label: string }` | When a node is selected. |

## Accessibility

### Keyboard Navigation

| Key | Action |
|---|---|
| `ArrowDown` | Move focus to the next item |
| `ArrowLeft` | Move focus backward or decrease value |
| `ArrowRight` | Move focus forward or increase value |
| `ArrowUp` | Move focus to the previous item |
| `Enter` | Activate or select the focused item |
| `Space` | Activate or select the focused item |

### ARIA

**Roles:** `tree`, `treeitem`

**Attributes:** `aria-disabled`, `aria-expanded`, `aria-selected`
