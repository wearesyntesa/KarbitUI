## Props

| Property | Attribute | Type | Default | Description |
|---|---|---|---|---|
| `variant` | `variant` | `TagVariant` | `'subtle'` | Visual variant controlling fill, border, and text styles. |
| `size` | `size` | `TagSize` | `'md'` | Tag size affecting font size and padding. |
| `colorScheme` | `color-scheme` | `ColorScheme` | — | Color scheme for the tag. Overrides default variant colors. |
| `closable` | `closable` | `boolean` | `false` | Show a close button that dismisses the tag with animation. |
| `interactive` | `interactive` | `boolean` | `false` | Make the tag focusable and clickable, emitting `kb-click`. |
| `dot` | `dot` | `boolean` | `false` | Show a small colored dot indicator before the label. |
| `kbDraggable` | `kb-draggable` | `boolean` | `false` | Enable HTML5 drag-and-drop on this tag. |
| `value` | `value` | `string` | `""` | Value identifier used for drag data and reorder events. |

## Slots

| Name | Description |
|---|---|
| (default) | Default content |
| `icon` | Leading icon element rendered before the label. |

## Events

| Event | Detail | Description |
|---|---|---|
| `kb-click` | none | Interactive tag clicked/activated. |
| `kb-close` | `{ source?: string }` or none | Close button clicked; tag animates out and removes itself. |
