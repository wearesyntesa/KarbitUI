## Props

| Property | Attribute | Type | Default | Description |
|---|---|---|---|---|
| `interactive` | `interactive` | `boolean \| undefined` | — | Override parent list `interactive` setting for this item. Falls back to parent `<kb-list>` value when `undefined`. |
| `divider` | `divider` | `boolean \| undefined` | — | Override parent list `dividers` setting for this item. Falls back to parent `<kb-list>` value when `undefined`. |
| `disabled` | `disabled` | `boolean` | `false` | Disables the item, preventing interaction and applying reduced opacity. |
| `href` | `href` | `string` | `""` | URL to navigate to - renders the item as an anchor element. |
| `target` | `target` | `string` | `""` | Link target attribute, e.g. `'_blank'`. Only used when `href` is set. |

## Slots

| Name | Description |
|---|---|
| (default) | Default content |
| `icon` | Leading icon or visual element. |
| `secondary` | Secondary description text below or beside the primary content. |
| `trailing` | Trailing content (badge, arrow, etc.) aligned to the right. |

## Events

| Event | Detail | Description |
|---|---|---|
| `kb-click` | none | Dispatched when an interactive item is clicked or activated via keyboard. |

## Accessibility
