## Props

| Property | Attribute | Type | Default | Description |
|---|---|---|---|---|
| `reorderable` | `reorderable` | `boolean` | `false` | Enable drag-and-drop reordering of child `kb-tag[kb-draggable]` elements. |
| `gap` | `gap` | `TagGroupGap` | `'md'` | Spacing between tags. |

## Slots

| Name | Description |
|---|---|
| (default) | Default content |

## Events

| Event | Detail | Description |
|---|---|---|
| `kb-reorder` | `{ order: readonly string[] }` | Fired after drag reorder completes. `detail.order` is an array of tag `value` strings. |
| `kb-drag-start` | `{ value: string }` | Fired when a tag drag begins. `detail.value` is the dragged tag value. |
| `kb-drag-end` | `{ value: string }` | Fired when a tag drag ends. `detail.value` is the dragged tag value. |
