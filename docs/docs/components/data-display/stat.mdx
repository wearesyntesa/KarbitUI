## Props

| Property | Attribute | Type | Default | Description |
|---|---|---|---|---|
| `label` | `label` | `string` | `""` | Descriptive label above the value. |
| `value` | `value` | `string` | `""` | The main statistic value to display. |
| `helpText` | `help-text` | `string` | — | Contextual help text displayed below the value (e.g. percentage change). |
| `indicator` | `indicator` | `StatIndicator` | `'none'` | Change direction indicator arrow. |
| `size` | `size` | `ComponentSize` | `'md'` | Stat size. |

## Slots

| Name | Description |
|---|---|
| `icon` | Optional icon displayed before the value. |
