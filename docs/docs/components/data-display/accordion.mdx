## Props

| Property | Attribute | Type | Default | Description |
|---|---|---|---|---|
| `open` | `open` | `boolean` | `false` | Whether the panel is expanded. Reflected to attribute for CSS targeting. |
| `disabled` | `disabled` | `boolean` | `false` | Prevents toggling and applies disabled styles to the trigger button. |

## Slots

| Name | Description |
|---|---|
| (default) | Default content |
| `trigger` | Clickable header that toggles the panel. |

## Events

| Event | Detail | Description |
|---|---|---|
| `kb-toggle` | `{ open: boolean }` | Dispatched when open state changes, with `detail.open`. |
