## Props

| Property | Attribute | Type | Default | Description |
|---|---|---|---|---|
| `variant` | `variant` | `TableVariant` | `'simple'` | Visual variant - `'simple'` (default lines), `'striped'` (alternating row bg), or `'bordered'` (cell borders). |
| `size` | `size` | `TableSize` | `'md'` | Cell and header padding size. |
| `hoverable` | `hoverable` | `boolean` | `true` | Highlight rows on hover. |
| `interactive` | `interactive` | `boolean` | `false` | Make body rows clickable/focusable and emit `kb-row-click` events. |
| `stickyHeader` | `sticky-header` | `boolean` | `false` | Stick the `<thead>` to the top of the scroll container. |
| `caption` | `caption` | `string` | `""` | Caption text shown in a labeled bar above the table. |
| `sortable` | `sortable` | `boolean` | `false` | Enable click-to-sort on header cells (use `data-no-sort` on a `<th>` to opt out). |
| `searchable` | `searchable` | `boolean` | `false` | Show a search toolbar that filters rows by text content. |
| `resizable` | `resizable` | `boolean` | `false` | Enable column-edge drag handles for width adjustment. |
| `searchPlaceholder` | `search-placeholder` | `string` | `'Search...'` | Placeholder text shown in the search input. |
| `emptyText` | `empty-text` | `string` | `'No results'` | Text shown when no rows match the search query (or the table has no rows). |

## Slots

| Name | Description |
|---|---|
| (default) | Default content |

## Events

| Event | Detail | Description |
|---|---|---|
| `kb-row-click` | `{ index: number, row: HTMLTableRowElement }` | Interactive body row clicked/activated. `detail: { index: number; row: HTMLTableRowElement }`. |
| `kb-sort` | `{ column: number, direction: SortDirection }` | Sort state changed. `detail: { column: number; direction: 'asc' \| 'desc' }`. |
