## Props

| Property | Attribute | Type | Default | Description |
|---|---|---|---|---|
| `value` | `value` | `string` | `'#000000ff'` | Current color value as hex (6 or 8 char). |
| `name` | `name` | `string` | — | Form field name. |
| `disabled` | `disabled` | `boolean` | `false` | Disable interaction. |
| `invalid` | `invalid` | `boolean` | `false` | Apply error styling. |
| `clearable` | `clearable` | `boolean` | `false` | Show clear button. |
| `presets` | `presets` | `string[]` | — | Preset colors for quick pick. |
| `showAlpha` | `show-alpha` | `boolean` | `true` | Show alpha channel controls. |
| `placeholder` | `placeholder` | `string` | `'Pick a color'` | Placeholder text. |
| `variant` | `variant` | `FormVariant` | `'outline'` | Visual variant. |
| `size` | `size` | `ComponentSize` | `'md'` | Component size. |

## Events

| Event | Detail | Description |
|---|---|---|
| `kb-change` | `KbChangeDetail` — discriminated union, narrow via `detail.source` | Value changed. Detail: `{ source: 'color-picker', value: string }`. |
| `kb-color-change` | `{ hex: string, r: number, g: number, b: number, a: number }` | Color changed. Detail: `{ hex, r, g, b, a }`. |
| `kb-clear` | `{ source?: string }` or none | Cleared. |
| `kb-open` | `{ source?: string }` or none | Panel opened. |
| `kb-close` | `{ source?: string }` or none | Panel closed. |

## Accessibility

### Keyboard Navigation

| Key | Action |
|---|---|
| `ArrowDown` | Move focus to the next item |
| `Enter` | Activate or select the focused item |
| `Escape` | Close or cancel the current action |
| `Space` | Activate or select the focused item |

### ARIA

**Roles:** `combobox`

**Attributes:** `aria-controls`, `aria-disabled`, `aria-expanded`, `aria-haspopup`, `aria-invalid`, `aria-label`
