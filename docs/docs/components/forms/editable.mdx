## Props

| Property | Attribute | Type | Default | Description |
|---|---|---|---|---|
| `value` | `value` | `string` | `""` | Current text value. |
| `placeholder` | `placeholder` | `string` | `'Click to edit'` | Placeholder shown when value is empty. |
| `disabled` | `disabled` | `boolean` | `false` | Disable interaction. |
| `invalid` | `invalid` | `boolean` | `false` | Mark as invalid — shows red underline. |
| `startWithEditView` | `start-with-edit-view` | `boolean` | `false` | Start in edit mode. |
| `activation` | `activation` | `'click' \| 'dblclick'` | `'click'` | Activation trigger. |
| `size` | `size` | `ComponentSize` | `'md'` | Display size. |
| `submitOnBlur` | `submit-on-blur` | `boolean` | `true` | Auto-submit when input loses focus. |
| `name` | `name` | `string` | — | Form field name. |

## Events

| Event | Detail | Description |
|---|---|---|
| `kb-change` | `KbChangeDetail` — discriminated union, narrow via `detail.source` | Fires when the value changes. Detail: `{ source: 'editable', value: string }`. |
| `kb-edit-submit` | `{ value: string, previousValue: string }` | Fires when editing is confirmed. Detail: `{ value: string, previousValue: string }`. |
| `kb-edit-cancel` | `{ value: string }` | Fires when editing is cancelled. Detail: `{ value: string }`. |

## Accessibility

### Keyboard Navigation

| Key | Action |
|---|---|
| `Enter` | Activate or select the focused item |
| `Escape` | Close or cancel the current action |
| `Space` | Activate or select the focused item |

### ARIA

**Roles:** `button`

**Attributes:** `aria-label`
