## Props

| Property | Attribute | Type | Default | Description |
|---|---|---|---|---|
| `value` | `value` | `number` | `0` | Current rating value. |
| `max` | `max` | `number` | `5` | Maximum number of stars. |
| `allowHalf` | `allow-half` | `boolean` | `false` | Allow half-star increments. |
| `size` | `size` | `ComponentSize` | `'md'` | Display size. |
| `disabled` | `disabled` | `boolean` | `false` | Disable interaction. |
| `readonly` | `readonly` | `boolean` | `false` | Read-only display (shows value but prevents changes). |
| `name` | `name` | `string` | — | Form field name. |

## Events

| Event | Detail | Description |
|---|---|---|
| `kb-change` | `KbChangeDetail` — discriminated union, narrow via `detail.source` | Fires when the rating value changes. Detail: `{ source: 'rating', value: string }`. |
| `kb-rating-hover` | `{ value: number }` | Fires during hover with the previewed value. Detail: `{ value: number }`. |

## Accessibility

### Keyboard Navigation

| Key | Action |
|---|---|
| `ArrowDown` | Move focus to the next item |
| `ArrowLeft` | Move focus backward or decrease value |
| `ArrowRight` | Move focus forward or increase value |
| `ArrowUp` | Move focus to the previous item |
| `End` | Move focus to the last item |
| `Home` | Move focus to the first item |

### ARIA

**Roles:** `slider`

**Attributes:** `aria-disabled`, `aria-label`, `aria-readonly`, `aria-valuemax`, `aria-valuemin`, `aria-valuenow`
