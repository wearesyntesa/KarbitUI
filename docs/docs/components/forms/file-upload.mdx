## Props

| Property | Attribute | Type | Default | Description |
|---|---|---|---|---|
| `accept` | `accept` | `string` | — | Accepted file types (MIME types or extensions). |
| `multiple` | `multiple` | `boolean` | `false` | Allow multiple file selection. |
| `maxSize` | `max-size` | `number` | `0` | Maximum file size in bytes. 0 = unlimited. |
| `maxFiles` | `max-files` | `number` | `0` | Maximum number of files. 0 = unlimited. |
| `size` | `size` | `ComponentSize` | `'md'` | Display size. |
| `disabled` | `disabled` | `boolean` | `false` | Disable interaction. |
| `invalid` | `invalid` | `boolean` | `false` | Apply error styling. |
| `name` | `name` | `string` | — | Form field name. |

## Events

| Event | Detail | Description |
|---|---|---|
| `kb-file-select` | `{ files: readonly File[] }` | Fires when files are selected. Detail: `{ files: File[] }`. |
| `kb-file-remove` | `{ name: string, index: number }` | Fires when a file is removed. Detail: `{ name: string, index: number }`. |
