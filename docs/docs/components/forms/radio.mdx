## Props

| Property | Attribute | Type | Default | Description |
|---|---|---|---|---|
| `size` | `size` | `ComponentSize` | `'md'` | Radio button size controlling circle dimensions, label text, and dot size. |
| `checked` | `checked` | `boolean` | `false` | Whether this radio is selected. Reflects to the `checked` attribute. |
| `disabled` | `disabled` | `boolean` | `false` | Disable interaction and apply dimmed styling. |
| `invalid` | `invalid` | `boolean` | `false` | Apply error border styling when unchecked. |
| `colorScheme` | `color-scheme` | `ColorScheme` | — | Color scheme for the selected state border and dot fill. |
| `name` | `name` | `string` | — | Form field name. Should match across radios in the same group. |
| `value` | `value` | `string` | `""` | Value emitted in `kb-change` event detail when this radio is selected. |
| `required` | `required` | `boolean` | `false` | Mark this radio as required for form validation. |

## Slots

| Name | Description |
|---|---|
| (default) | Default content |
| `description` | Helper text rendered below the label. |

## Events

| Event | Detail | Description |
|---|---|---|
| `kb-change` | `KbChangeDetail` — discriminated union, narrow via `detail.source` | Checked state changed. Detail: `{ checked: boolean, value: string }`. |
| `kb-focus` | `{ source?: string }` or none | — |
| `kb-blur` | `{ source?: string }` or none | — |
