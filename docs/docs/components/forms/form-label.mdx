## Props

| Property | Attribute | Type | Default | Description |
|---|---|---|---|---|
| `for` | `for` | `string` | — | `for` attribute linking the label to a form element by ID. |
| `size` | `size` | `ComponentSize` | `'md'` | Label size controlling font size. |
| `required` | `required` | `boolean` | `false` | Show a red asterisk after the label text. |
| `optional` | `optional` | `boolean` | `false` | Show "(optional)" after the label text. Hidden when `required` is true. |

## Slots

| Name | Description |
|---|---|
| (default) | Default content |
| `info` | Inline help icon or text rendered after the label. |
