## Props

| Property | Attribute | Type | Default | Description |
|---|---|---|---|---|
| `inputId` | `input-id` | `string` | — | Forwarded `id` applied to the native `<textarea>` element. Allows external `<label for="...">` to work. |
| `variant` | `variant` | `FormVariant` | `'outline'` | Form textarea visual variant. |
| `size` | `size` | `ComponentSize` | `'md'` | Textarea size controlling padding, font size, and action sizing. |
| `placeholder` | `placeholder` | `string` | — | Placeholder text shown when the textarea is empty. |
| `value` | `value` | `string` | `""` | Current textarea value. Two-way bindable. |
| `name` | `name` | `string` | — | Form field name, used in form submissions. |
| `rows` | `rows` | `number` | `4` | Initial visible row count. |
| `maxLength` | `max-length` | `number` | — | Maximum character count. Enables the character counter when set. |
| `resize` | `resize` | `ResizeMode` | `'vertical'` | CSS resize behavior. Overridden to `'none'` when `autoResize` is enabled. |
| `disabled` | `disabled` | `boolean` | `false` | Disable interaction and apply dimmed styling. |
| `invalid` | `invalid` | `boolean` | `false` | Mark the textarea as invalid with error border styling. |
| `readonly` | `readonly` | `boolean` | `false` | Make the textarea read-only (focusable but not editable). |
| `required` | `required` | `boolean` | `false` | Mark the textarea as required for form validation. |
| `clearable` | `clearable` | `boolean` | `false` | Show a clear button when the textarea has a value. |
| `loading` | `loading` | `boolean` | `false` | Show a loading spinner in the action area. |
| `autoResize` | `auto-resize` | `boolean` | `false` | Automatically grow the textarea height to fit content. Disables manual resizing. |

## Events

| Event | Detail | Description |
|---|---|---|
| `kb-input` | `{ value: string }` | Value changed on input. Detail: `{ value: string }`. |
| `kb-change` | `KbChangeDetail` — discriminated union, narrow via `detail.source` | Value committed on change. Detail: `{ value: string }`. |
| `kb-focus` | `{ source?: string }` or none | Textarea focused. |
| `kb-blur` | `{ source?: string }` or none | Textarea blurred. |
| `kb-clear` | `{ source?: string }` or none | Clear button clicked (when clearable). |
