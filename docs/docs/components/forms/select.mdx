Karbit `Select` is a custom combobox/listbox component, not a thin wrapper over native `<select>` / `<option>` markup.

- use the `options` array API intentionally
- use `:options.prop` when you render raw `kb-select` in Vue
- use the official Vue wrapper when you want normal prop binding and `v-model:value`

```vue
<script setup lang="ts">
import { Select } from '@wearesyntesa/karbit-ui/vue';
import { ref } from 'vue';

const role = ref('');
const options = [
  { value: 'admin', label: 'Admin' },
  { value: 'editor', label: 'Editor' },
  { value: 'viewer', label: 'Viewer' },
];
</script>

<template>
  <Select v-model:value="role" :options="options" placeholder="Role" />
</template>
```

## Props

| Property | Attribute | Type | Default | Description |
|---|---|---|---|---|
| `variant` | `variant` | `FormVariant` | `'outline'` | Form select visual variant. |
| `size` | `size` | `ComponentSize` | `'md'` | Select size controlling padding, font size, and icon sizing. |
| `placeholder` | `placeholder` | `string` | — | Placeholder text shown when no value is selected. |
| `value` | `value` | `V \| ''` | `""` | Currently selected value. |
| `name` | `name` | `string` | — | Form field name, used in form submissions. |
| `options` | `options` | `readonly SelectOptionOrGroup<V>[]` | `[]` | Array of options or option groups to render in the dropdown. |
| `disabled` | `disabled` | `boolean` | `false` | Disable interaction and apply dimmed styling. |
| `invalid` | `invalid` | `boolean` | `false` | Mark the select as invalid with error border styling. |
| `required` | `required` | `boolean` | `false` | Mark the select as required for form validation. |
| `clearable` | `clearable` | `boolean` | `false` | Show a clear button when a value is selected. |
| `loading` | `loading` | `boolean` | `false` | Show a loading spinner in place of the chevron. |
| `emptyText` | `empty-text` | `string` | `'No results'` | Text shown in the dropdown panel when there are no options. |

## Slots

| Name | Description |
|---|---|
| `icon` | Leading icon element inside the select wrapper. |

## Events

| Event | Detail | Description |
|---|---|---|
| `kb-change` | `KbChangeDetail` — discriminated union, narrow via `detail.source` | Value changed. Detail: `{ value: string }`. |
| `kb-focus` | `{ source?: string }` or none | Select focused. |
| `kb-blur` | `{ source?: string }` or none | Select blurred. |
| `kb-clear` | `{ source?: string }` or none | Clear button clicked (when clearable). |

## Accessibility

### Keyboard Navigation

| Key | Action |
|---|---|
| `ArrowDown` | Move focus to the next item |
| `ArrowUp` | Move focus to the previous item |
| `End` | Move focus to the last item |
| `Enter` | Activate or select the focused item |
| `Escape` | Close or cancel the current action |
| `Home` | Move focus to the first item |
| `Space` | Activate or select the focused item |
| `Tab` | Move focus to the next focusable element |
