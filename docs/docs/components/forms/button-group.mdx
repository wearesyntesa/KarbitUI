## Props

| Property | Attribute | Type | Default | Description |
|---|---|---|---|---|
| `direction` | `direction` | `ButtonGroupDirection` | `'horizontal'` | Layout direction of the grouped buttons. |
| `spaced` | `spaced` | `boolean` | `false` | Use spaced mode (gap between buttons) instead of attached mode (shared borders). |
| `label` | `label` | `string` | `""` | Accessible label for the button group (required for screen readers). |
