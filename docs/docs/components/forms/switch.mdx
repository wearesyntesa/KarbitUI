## Props

| Property | Attribute | Type | Default | Description |
|---|---|---|---|---|
| `size` | `size` | `ComponentSize` | `'md'` | Switch size controlling track, thumb, and label dimensions. |
| `checked` | `checked` | `boolean` | `false` | Whether the switch is on. Reflects to the `checked` attribute. |
| `disabled` | `disabled` | `boolean` | `false` | Disable interaction and apply dimmed styling. |
| `invalid` | `invalid` | `boolean` | `false` | Apply error border styling when unchecked. |
| `loading` | `loading` | `boolean` | `false` | Show a loading spinner inside the thumb and disable interaction. |
| `showIcons` | `show-icons` | `boolean` | `false` | Show check/cross icons inside the thumb. |
| `colorScheme` | `color-scheme` | `ColorScheme` | — | Color scheme for the checked state track, icon, and hover border. |
| `labelPosition` | `label-position` | `SwitchLabelPosition` | `'right'` | Position of the label relative to the switch track. |
| `name` | `name` | `string` | — | Form field name, used in form submissions. |

## Slots

| Name | Description |
|---|---|
| (default) | Default content |
| `description` | Helper text rendered below the label. |

## Events

| Event | Detail | Description |
|---|---|---|
| `kb-change` | `KbChangeDetail` — discriminated union, narrow via `detail.source` | Checked state changed. Detail: `{ checked: boolean }`. |
| `kb-focus` | `{ source?: string }` or none | — |
| `kb-blur` | `{ source?: string }` or none | — |
