## Props

| Property | Attribute | Type | Default | Description |
|---|---|---|---|---|
| `length` | `length` | `number` | `4` | Number of input fields. |
| `type` | `type` | `PinInputType` | `'numeric'` | Input type constraint. |
| `mask` | `mask` | `boolean` | `false` | Show dots instead of characters. |
| `size` | `size` | `ComponentSize` | `'md'` | Input size controlling cell dimensions and font size. |
| `disabled` | `disabled` | `boolean` | `false` | Disable interaction. |
| `invalid` | `invalid` | `boolean` | `false` | Apply error border styling. |
| `name` | `name` | `string` | — | Form field name. |
| `value` | `value` | `string` | `""` | Current combined value. |

## Events

| Event | Detail | Description |
|---|---|---|
| `kb-input` | `{ value: string }` | Fires on every character change. Detail: `{ value: string }`. |
| `kb-change` | `KbChangeDetail` — discriminated union, narrow via `detail.source` | Fires on value commit. Detail: `{ source: 'pin-input', value: string }`. |
| `kb-complete` | `{ value: string }` | Fires when all inputs are filled. Detail: `{ value: string }`. |

## Accessibility

### Keyboard Navigation

| Key | Action |
|---|---|
| `ArrowLeft` | Move focus backward or decrease value |
| `ArrowRight` | Move focus forward or increase value |
| `Backspace` | Delete the previous character or item |

### ARIA

**Roles:** `group`

**Attributes:** `aria-invalid`, `aria-label`
