## Props

| Property | Attribute | Type | Default | Description |
|---|---|---|---|---|
| `value` | `value` | `string` | `""` | Current value in YYYY-MM-DD format. |
| `placeholder` | `placeholder` | `string` | — | Placeholder text. |
| `min` | `min` | `string` | — | Earliest selectable date (YYYY-MM-DD). |
| `max` | `max` | `string` | — | Latest selectable date (YYYY-MM-DD). |
| `variant` | `variant` | `FormVariant` | `'outline'` | Visual variant. |
| `size` | `size` | `ComponentSize` | `'md'` | Display size. |
| `disabled` | `disabled` | `boolean` | `false` | Disable interaction. |
| `invalid` | `invalid` | `boolean` | `false` | Apply error styling. |
| `clearable` | `clearable` | `boolean` | `false` | Show clear button. |
| `name` | `name` | `string` | — | Form field name. |

## Events

| Event | Detail | Description |
|---|---|---|
| `kb-change` | `KbChangeDetail` — discriminated union, narrow via `detail.source` | Value changed. Detail: `{ source: 'date-picker', value: string }`. |
| `kb-focus` | `{ source?: string }` or none | Focused. |
| `kb-blur` | `{ source?: string }` or none | Blurred. |
| `kb-clear` | `{ source?: string }` or none | Cleared. |

## Accessibility

### Keyboard Navigation

| Key | Action |
|---|---|
| `ArrowDown` | Move focus to the next item |
| `ArrowLeft` | Move focus backward or decrease value |
| `ArrowRight` | Move focus forward or increase value |
| `ArrowUp` | Move focus to the previous item |
| `Enter` | Activate or select the focused item |
| `Escape` | Close or cancel the current action |
| `Space` | Activate or select the focused item |

### ARIA

**Roles:** `combobox`, `grid`, `gridcell`

**Attributes:** `aria-controls`, `aria-disabled`, `aria-expanded`, `aria-haspopup`, `aria-invalid`, `aria-label`, `aria-selected`
