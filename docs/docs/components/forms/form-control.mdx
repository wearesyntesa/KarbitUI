# Form Control

A wrapper that provides label, helper text, and error messaging for form fields.

## Props

| Property | Attribute | Type | Default | Description |
|---|---|---|---|---|
| `required` | `required` | `boolean` | `false` | Mark the field as required. Propagated to child input and label. |
| `invalid` | `invalid` | `boolean` | `false` | Mark the field as invalid. Propagated to child input; switches helper → error text. |
| `disabled` | `disabled` | `boolean` | `false` | Disable the field. Propagated to child input. |

## Slots

| Name | Description |
|---|---|
| (default) | Default content |
| `helper` | Muted helper text below the input (hidden when `invalid` with error slot). |
| `error` | Red error text below the input (shown only when `invalid`). |
| `counter` | Right-aligned text below the input (e.g. character count "12/100"). |

## Accessibility

### ARIA

**Roles:** `group`

**Attributes:** `aria-labelledby`, `aria-live`
