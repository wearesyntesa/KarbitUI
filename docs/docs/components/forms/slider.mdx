## Props

| Property | Attribute | Type | Default | Description |
|---|---|---|---|---|
| `value` | `value` | `number` | `50` | Current slider value. |
| `min` | `min` | `number` | `0` | Minimum value. |
| `max` | `max` | `number` | `100` | Maximum value. |
| `step` | `step` | `number` | `1` | Step increment. |
| `size` | `size` | `ComponentSize` | `'md'` | Slider size. |
| `disabled` | `disabled` | `boolean` | `false` | Disable interaction. |
| `readonly` | `readonly` | `boolean` | `false` | Prevent value changes but allow focus. |
| `colorScheme` | `color-scheme` | `ColorScheme` | — | Color scheme for the filled track and thumb. |
| `name` | `name` | `string` | — | Form field name. |
| `showValue` | `show-value` | `boolean` | `false` | Show value tooltip on hover and drag. |
| `showTicks` | `show-ticks` | `boolean` | `false` | Show tick marks at step intervals (max 20 ticks). |
| `showRange` | `show-range` | `boolean` | `false` | Show min/max labels below the track. |
| `formatValue` | — (JS only) | `(value: number)` | `> string` | Custom value formatter for tooltip display. |

## Events

| Event | Detail | Description |
|---|---|---|
| `kb-change` | `KbChangeDetail` — discriminated union, narrow via `detail.source` | Value changed. Detail: `{ source: 'slider', value: string }`. |
| `kb-input` | `{ value: string }` | Value changing during drag. Detail: `{ value: string }`. |

## Accessibility

### Keyboard Navigation

| Key | Action |
|---|---|
| `ArrowDown` | Move focus to the next item |
| `ArrowLeft` | Move focus backward or decrease value |
| `ArrowRight` | Move focus forward or increase value |
| `ArrowUp` | Move focus to the previous item |
| `End` | Move focus to the last item |
| `Home` | Move focus to the first item |
| `PageDown` | Decrease by a large step |
| `PageUp` | Increase by a large step |

### ARIA

**Roles:** `slider`

**Attributes:** `aria-disabled`, `aria-readonly`, `aria-valuemax`, `aria-valuemin`, `aria-valuenow`, `aria-valuetext`
