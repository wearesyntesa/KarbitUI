## Props

| Property | Attribute | Type | Default | Description |
|---|---|---|---|---|
| `value` | `value` | `number` | `0` | Current numeric value. |
| `min` | `min` | `number` | — | Minimum allowed value. |
| `max` | `max` | `number` | — | Maximum allowed value. |
| `step` | `step` | `number` | `1` | Step increment for stepper buttons and arrow keys. |
| `size` | `size` | `ComponentSize` | `'md'` | Input size controlling padding and font size. |
| `variant` | `variant` | `FormVariant` | `'outline'` | Form input visual variant. |
| `disabled` | `disabled` | `boolean` | `false` | Disable interaction. |
| `invalid` | `invalid` | `boolean` | `false` | Apply error border styling. |
| `name` | `name` | `string` | — | Form field name. |

## Events

| Event | Detail | Description |
|---|---|---|
| `kb-change` | `KbChangeDetail` — discriminated union, narrow via `detail.source` | Value committed. Detail: `{ source: 'number-input', value: string }`. |
| `kb-input` | `{ value: string }` | Value changed during input. Detail: `{ value: string }`. |

## Accessibility

### Keyboard Navigation

| Key | Action |
|---|---|
| `ArrowDown` | Move focus to the next item |
| `ArrowUp` | Move focus to the previous item |

### ARIA

**Roles:** `spinbutton`

**Attributes:** `aria-invalid`, `aria-label`, `aria-valuemax`, `aria-valuemin`, `aria-valuenow`
