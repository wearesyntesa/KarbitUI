## Props

| Property | Attribute | Type | Default | Description |
|---|---|---|---|---|
| `value` | `value` | `V \| ''` | `""` | Currently selected value. |
| `options` | `options` | `readonly SegmentOption<V>[]` | `[]` | Options to render as segments. |
| `size` | `size` | `ComponentSize` | `'md'` | Display size. |
| `disabled` | `disabled` | `boolean` | `false` | Disable all segments. |
| `name` | `name` | `string` | — | Form field name. |

## Events

| Event | Detail | Description |
|---|---|---|
| `kb-change` | `KbChangeDetail` — discriminated union, narrow via `detail.source` | Fires when the selected value changes. Detail: `{ source: 'segment', value: string }`. |

## Accessibility

### Keyboard Navigation

| Key | Action |
|---|---|
| `ArrowDown` | Move focus to the next item |
| `ArrowLeft` | Move focus backward or decrease value |
| `ArrowRight` | Move focus forward or increase value |
| `ArrowUp` | Move focus to the previous item |
| `End` | Move focus to the last item |
| `Home` | Move focus to the first item |

### ARIA

**Roles:** `radio`, `radiogroup`

**Attributes:** `aria-checked`, `aria-disabled`, `aria-label`
