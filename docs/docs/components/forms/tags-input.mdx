## Props

| Property | Attribute | Type | Default | Description |
|---|---|---|---|---|
| `values` | `values` | `readonly string[]` | `[]` | Current tag values. |
| `placeholder` | `placeholder` | `string` | — | Placeholder text for the input. |
| `variant` | `variant` | `FormVariant` | `'outline'` | Visual variant. |
| `size` | `size` | `ComponentSize` | `'md'` | Display size. |
| `disabled` | `disabled` | `boolean` | `false` | Disable interaction. |
| `invalid` | `invalid` | `boolean` | `false` | Apply error styling. |
| `max` | `max` | `number` | `0` | Maximum number of tags allowed. 0 = unlimited. |
| `allowDuplicates` | `allow-duplicates` | `boolean` | `false` | Allow duplicate tag values. |
| `name` | `name` | `string` | — | Form field name. |

## Events

| Event | Detail | Description |
|---|---|---|
| `kb-change` | `KbChangeDetail` — discriminated union, narrow via `detail.source` | Fires on any value change. Detail: `{ source: 'tags-input', values: string[] }`. |
| `kb-tag-add` | `{ value: string }` | Fires when a tag is added. Detail: `{ value: string }`. |
| `kb-tag-remove` | `{ value: string, index: number }` | Fires when a tag is removed. Detail: `{ value: string, index: number }`. |
