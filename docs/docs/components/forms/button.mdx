## Props

| Property | Attribute | Type | Default | Description |
|---|---|---|---|---|
| `variant` | `variant` | `ButtonVariant` | `'solid'` | Visual variant controlling border, background, and text styles. |
| `size` | `size` | `ButtonSize` | `'md'` | Button size controlling padding, font size, and icon sizing. |
| `colorScheme` | `color-scheme` | `ColorScheme` | — | Color scheme override. When unset, the variant's default color is used. |
| `disabled` | `disabled` | `boolean` | `false` | Disable interaction and apply dimmed styling. |
| `loading` | `loading` | `boolean` | `false` | Show a spinner and disable interaction. |
| `loadingText` | `loading-text` | `string` | — | Text displayed next to the spinner when loading. When omitted, the label is hidden during loading. |
| `type` | `type` | `'button' \| 'submit' \| 'reset'` | `'button'` | HTML button type attribute. |
| `fullWidth` | `full-width` | `boolean` | `false` | Stretch the button to fill its container width. |

## Slots

| Name | Description |
|---|---|
| (default) | Default content |
| `icon-left` | Leading icon element (SVG recommended). |
| `icon-right` | Trailing icon element (SVG recommended). |
