## Props

| Property | Attribute | Type | Default | Description |
|---|---|---|---|---|
| `direction` | `direction` | `Orientation` | `'vertical'` | Layout direction of the grouped radios. |
| `name` | `name` | `string` | — | Form field name propagated to child radios that don't set their own. |
| `size` | `size` | `ComponentSize` | — | Size propagated to child radios that don't set their own size. |
| `colorScheme` | `color-scheme` | `ColorScheme` | — | Color scheme propagated to child radios that don't set their own. |
| `value` | `value` | `string` | — | Currently selected value. Reflects the checked radio's value. Can be set programmatically. |
| `required` | `required` | `boolean` | `false` | Mark the group as required — propagated to child radios and exposed as `aria-required` on the group. |

## Slots

| Name | Description |
|---|---|
| (default) | Default content |
| `label` | Group label rendered above the radios. |

## Events

| Event | Detail | Description |
|---|---|---|
| `kb-change` | `KbChangeDetail` — discriminated union, narrow via `detail.source` | Selection changed. Detail: `{ value: string }`. |

## Accessibility

### Keyboard Navigation

| Key | Action |
|---|---|
| `ArrowDown` | Move focus to the next item |
| `ArrowLeft` | Move focus backward or decrease value |
| `ArrowRight` | Move focus forward or increase value |
| `ArrowUp` | Move focus to the previous item |

**Roles:** `radiogroup`

**Attributes:** `aria-labelledby`, `aria-required`
