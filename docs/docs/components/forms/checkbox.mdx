## Props

| Property | Attribute | Type | Default | Description |
|---|---|---|---|---|
| `size` | `size` | `ComponentSize` | `'md'` | Checkbox size controlling box dimensions, label text, and icon size. |
| `checked` | `checked` | `boolean` | `false` | Whether the checkbox is checked. Reflects to the `checked` attribute. |
| `disabled` | `disabled` | `boolean` | `false` | Disable interaction and apply dimmed styling. |
| `indeterminate` | `indeterminate` | `boolean` | `false` | Show a horizontal dash icon instead of a checkmark (third state). |
| `invalid` | `invalid` | `boolean` | `false` | Apply error border styling when unchecked. |
| `colorScheme` | `color-scheme` | `ColorScheme` | — | Color scheme for the checked/indeterminate state background and icon. |
| `name` | `name` | `string` | — | Form field name, used in form submissions. |
| `value` | `value` | `string` | — | Value included in `kb-change` event detail and form submissions. |

## Slots

| Name | Description |
|---|---|
| (default) | Default content |
| `description` | Helper text rendered below the label. |

## Events

| Event | Detail | Description |
|---|---|---|
| `kb-change` | `KbChangeDetail` — discriminated union, narrow via `detail.source` | Checked state changed. Detail: `{ checked: boolean, value?: string }`. |
| `kb-focus` | `{ source?: string }` or none | — |
| `kb-blur` | `{ source?: string }` or none | — |

## Accessibility

### Keyboard Navigation

| Key | Action |
|---|---|
| `Space` | Activate or select the focused item |

### ARIA

**Roles:** `checkbox`

**Attributes:** `aria-checked`, `aria-invalid`
