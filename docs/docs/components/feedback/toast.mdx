## Props

| Property | Attribute | Type | Default | Description |
|---|---|---|---|---|
| `status` | `status` | `ToastStatus` | `'info'` | Semantic status controlling icon, border accent, and progress bar color. |
| `position` | `position` | `ToastPosition` | `'top-right'` | Screen position where the toast appears. |
| `duration` | `duration` | `number` | `5000` | Auto-dismiss delay in milliseconds. Pauses on hover. |
| `closable` | `closable` | `boolean` | `true` | Show a close button for manual dismissal. |
| `showIcon` | `show-icon` | `boolean` | `true` | Show the status icon before the message content. |

## Slots

| Name | Description |
|---|---|
| (default) | Default content |
| `title` | Optional bold title above the message. |
| `action` | Inline action area (buttons, links). |

## Events

| Event | Detail | Description |
|---|---|---|
| `kb-close` | `{ source?: string }` or none | Dispatched when the toast is dismissed. |
