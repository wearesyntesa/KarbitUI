## Props

| Property | Attribute | Type | Default | Description |
|---|---|---|---|---|
| `status` | `status` | `AlertStatus` | `'info'` | Semantic status controlling icon and color scheme. |
| `variant` | `variant` | `AlertVariant` | `'subtle'` | Visual variant controlling border, background, and text styles. |
| `size` | `size` | `AlertSize` | `'md'` | Alert size controlling padding and font size. |
| `closable` | `closable` | `boolean` | `false` | Show a close button that dismisses the alert with animation. |
| `showIcon` | `show-icon` | `boolean` | `true` | Show the status icon before the message content. |
| `duration` | `duration` | `number` | `0` | Auto-dismiss delay in milliseconds. `0` disables auto-dismiss. |

## Slots

| Name | Description |
|---|---|
| (default) | Default content |
| `title` | Bold heading rendered above the description. |
| `action` | Inline action area (buttons, links) rendered after content. |
| `detail` | Expandable detail content toggled via a chevron button. |
| `icon` | Custom icon override. |

## Events

| Event | Detail | Description |
|---|---|---|
| `kb-close` | `{ source?: string }` or none | Dispatched when the alert is dismissed (closable or auto-dismiss). |
| `kb-toggle` | `{ open: boolean }` | Dispatched when collapsible detail is expanded/collapsed. |

## Accessibility

### ARIA

**Roles:** `alert`

**Attributes:** `aria-expanded`, `aria-label`
