## Props

| Property | Attribute | Type | Default | Description |
|---|---|---|---|---|
| `value` | `value` | `number` | `0` | Current progress value between `min` and `max`. |
| `max` | `max` | `number` | `100` | Maximum progress value. |
| `min` | `min` | `number` | `0` | Minimum progress value. |
| `size` | `size` | `ComponentSize` | `'md'` | Size preset. |
| `indeterminate` | `indeterminate` | `boolean` | `false` | Show animated indeterminate spinner. |
| `showValue` | `show-value` | `boolean` | `false` | Show percentage label inside the ring. |
| `valueLabel` | `value-label` | `string` | — | Custom label text instead of computed percentage. |
| `colorScheme` | `color-scheme` | `ColorScheme` | — | Color scheme for the progress arc. |
| `thickness` | `thickness` | `number` | `1` | Ring thickness multiplier. |
