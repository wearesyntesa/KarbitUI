## Props

| Property | Attribute | Type | Default | Description |
|---|---|---|---|---|
| `value` | `value` | `number` | `0` | Current progress value between `min` and `max`. |
| `max` | `max` | `number` | `100` | Maximum progress value. |
| `min` | `min` | `number` | `0` | Minimum progress value. |
| `size` | `size` | `ProgressSize` | `'md'` | Bar height preset. |
| `indeterminate` | `indeterminate` | `boolean` | `false` | Show an animated indeterminate (looping) fill instead of a determinate bar. |
| `showValue` | `show-value` | `boolean` | `false` | Display the percentage or custom `valueLabel` next to the bar. |
| `valueLabel` | `value-label` | `string` | — | Custom text displayed instead of the computed percentage when `showValue` is true. |
| `colorScheme` | `color-scheme` | `ColorScheme` | — | Color scheme for the fill bar. When unset, defaults to blue (or auto-color thresholds). |
| `striped` | `striped` | `boolean` | `false` | Apply a diagonal stripe pattern to the fill bar. |
| `animated` | `animated` | `boolean` | `false` | Animate the stripe pattern. Requires `striped` to be true. |
| `segments` | `segments` | `number` | `0` | Number of visual segments dividing the track. `0` disables segmentation. |
| `autoColor` | `auto-color` | `boolean` | `false` | Automatically change fill color based on value thresholds (blue → yellow → red). |

## Slots

| Name | Description |
|---|---|
| `label` | Text label rendered above the progress bar. |
