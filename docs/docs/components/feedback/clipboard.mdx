## Props

| Property | Attribute | Type | Default | Description |
|---|---|---|---|---|
| `value` | `value` | `string` | `""` | The text value to copy to the clipboard. |
| `label` | `label` | `string` | `""` | Optional label shown next to the icon (button mode) or above the value (inline mode). |
| `copiedLabel` | `copied-label` | `string` | `'Copied'` | Label shown after a successful copy. |
| `showValue` | `show-value` | `boolean` | `false` | Display the value text inline with an integrated copy trigger. |
| `size` | `size` | `ComponentSize` | `'md'` | Component size. |
| `variant` | `variant` | `ClipboardVariant` | `'outline'` | Visual variant. |
| `disabled` | `disabled` | `boolean` | `false` | Disables interaction. |
| `timeout` | `timeout` | `number` | `1500` | Duration in ms to show the copied state. |

## Events

| Event | Detail | Description |
|---|---|---|
| `kb-copy` | `{ value: string, success: boolean }` | Dispatched after a copy attempt with `{ value, success }`. |
