## Props

| Property | Attribute | Type | Default | Description |
|---|---|---|---|---|
| `title` | `title` | `string` | `""` | Main title text. |
| `description` | `description` | `string` | `""` | Supporting description text. |
| `size` | `size` | `ComponentSize` | `'md'` | Size preset. |

## Slots

| Name | Description |
|---|---|
| (default) | Default content |
| `icon` | Icon or illustration. |
| `action` | Primary action button(s). |
