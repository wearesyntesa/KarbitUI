## Props

| Property | Attribute | Type | Default | Description |
|---|---|---|---|---|
| `variant` | `variant` | `BadgeVariant` | `'subtle'` | Visual variant controlling border, background, and text styles. |
| `size` | `size` | `BadgeSize` | `'sm'` | Badge size controlling padding, font size, and dot/close sizing. |
| `colorScheme` | `color-scheme` | `ColorScheme` | — | Color scheme override. When unset, the variant's default color is used. |
| `dot` | `dot` | `boolean` | `false` | Show a status dot before the badge label. |
| `pulse` | `pulse` | `boolean` | `false` | Animate the status dot with a ping effect. Requires `dot` to be true. |
| `ping` | `ping` | `boolean` | `false` | Show a notification ping indicator on the badge corner. |
| `interactive` | `interactive` | `boolean` | `false` | Make the badge clickable with hover/focus styles and `kb-click` events. |
| `closable` | `closable` | `boolean` | `false` | Show a close button that dismisses the badge with animation. |

## Slots

| Name | Description |
|---|---|
| (default) | Default content |
| `icon` | Leading icon element rendered before the label. |

## Events

| Event | Detail | Description |
|---|---|---|
| `kb-click` | none | Interactive badge clicked/activated. |
| `kb-close` | `{ source?: string }` or none | Close button clicked; badge animates out and removes itself. |
