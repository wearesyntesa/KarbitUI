## Props

| Property | Attribute | Type | Default | Description |
|---|---|---|---|---|
| `variant` | `variant` | `SpinnerVariant` | `'border'` | Spinner animation style. |
| `size` | `size` | `ComponentSize` | `'md'` | Spinner dimensions. |
| `colorScheme` | `color-scheme` | `ColorScheme` | — | Color scheme for the spinner stroke/fill. When unset, defaults to blue. |
| `label` | `label` | `string` | `'Loading...'` | Accessible label text, also used as visible text when `showLabel` is true. |
| `showLabel` | `show-label` | `boolean` | `false` | Display the label text below or beside the spinner. |
| `speed` | `speed` | `SpinnerSpeed` | `'normal'` | Animation speed. |
| `track` | `track` | `boolean` | `false` | Show a grey track ring behind the spinning border. Only applies to `'border'` variant. |
| `labelPosition` | `label-position` | `'bottom' \| 'right'` | `'bottom'` | Label position relative to the spinner. |
| `overlay` | `overlay` | `boolean` | `false` | Wrap slot content with a semi-transparent overlay and center the spinner above it. |

## Slots

| Name | Description |
|---|---|
| (default) | Default content |
