## Props

| Property | Attribute | Type | Default | Description |
|---|---|---|---|---|
| `variant` | `variant` | `SkeletonVariant` | `'pulse'` | Animation variant. |
| `size` | `size` | `ComponentSize` | `'md'` | Predefined height via size token. Overridden by explicit `height` prop. |
| `width` | `width` | `string` | `'100%'` | Explicit width (CSS value). |
| `height` | `height` | `string` | — | Explicit height (CSS value). Overrides `size` when set. |
| `circle` | `circle` | `boolean` | `false` | Render as a circle (equal width and height). |
| `lines` | `lines` | `number` | `1` | Number of text-like lines to render. When > 1, ignores width/height and renders stacked bars. |
